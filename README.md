# Google-img-fetcher
- fetch google image
- only with **preview pic**

# Use

## run 
`python main.py`

## setting
At line 10
```
# The path you save img
TARGET_PATH = "./img/"  

# How many do you want to fetch
ORDER_FETCH_NUM = 500

# Seacrh key word
KEY_WORD = '木柵動物園'
```

# Note
you may need install Beautiful Soup before use

`pip install beautifulsoup4`
